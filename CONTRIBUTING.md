# Contributing to Wunderspec

We welcome [discussions][] on Wunderspec features and tooling. We would listen
to your suggestions on the domain-specific language of Wunderspec, though we do
not promise you to implement any of the discussed features in this project.
Critical bugs that seriously affect users' workflow are also to be raised in
the discussions.

The main development is done by the Wunderspec team in a separate repository.
The AI tools have changed the dynamics of open source development. At the
moment, we are not looking for code contributions and opinionated issues.
High-level requests are welcome in [discussions][].

The source code of this Wunderspec core is provided to you under [Functional
Source License, Version 1.1][LICENSE], to become Apache 2.0 in the future. We
open the source code, so you can trust us that the code does not contain hidden
logic. When forking the project, respect this [LICENSE][].
